# Selenium Automation – Authentication Tests

## Overview
This project demonstrates automated testing for Signup, Login, Logout, and Account Deletion flows using Selenium WebDriver, Java, and TestNG.

The focus is on applying simple testing ideas to build clean, stable, and maintainable automation tests.

## Features
- Automated authentication flow (Signup → Login → Logout → Delete Account)
- Random generation of usernames and email addresses to avoid duplicate data issues
- Element locating using CSS Selectors
- Test execution control using TestNG priorities
- Repeating test cases using TestNG XML instead of invocationCount
- Implicit Wait to improve test stability

## Technologies Used
- Java  
- Selenium WebDriver  
- TestNG  
- EdgeDriver
